## ‚öôÔ∏è Setup & Installation

If you'd like to run this portfolio locally or fork it for your own use:

```bash
# Clone the repository
git clone https://github.com/AJSorbello/ajfolio.git

# Navigate to the project directory
cd ajfolio

# Install dependencies
npm install

# Start the development server
npm start

# The site will be available at http://localhost:3000
```

### Prerequisites
- Node.js (v16 or higher)
- npm (v7 or higher)

## üîÑ Deployment

This portfolio is configured for seamless deployment to Netlify:

1. Fork this repository
2. Connect your GitHub account to Netlify
3. Select the repository for deployment
4. Configure build settings:
   - Build command: `npm run build`
   - Publish directory: `build`
5. Deploy!
